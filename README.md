# Naneungolae(나는고래)
>일대일 익명 매칭을 통해 칭찬을 주고받는 앱<br>
>칭찬을 통해 얻은 먹이로 고래를 수집하고 성장시켜 보세요!

## 사용 기술
- SwiftUI
- Firebase
